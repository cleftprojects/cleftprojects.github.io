What is CleftProjects?

CleftProjects is an indie dev studio — a space where ideas are turned into working digital products. Not a big agency. Not a freelance side hustle. A studio with intent.

We look for the gaps — the unsolved problems, the overlooked needs, the tools that don't exist yet — and we build for them. Every project is deliberate. Every line of code has a reason.
