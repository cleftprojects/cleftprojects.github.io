# CleftProjects

> **Finding gaps. Building solutions.**

CleftProjects is an indie dev studio that builds purposeful digital products — not just to fulfill a brief, but to seek out unsolved problems and build solutions for them.

Not a big agency. Not a freelance side hustle. A studio with intent.

---

## 🛠️ Technologies

**Backend** — Python · Django · Django REST Framework · Django Testing · PostgreSQL

**Frontend** — HTML5 · CSS3 · JavaScript · HTMX · Tailwind CSS · Alpine.js

**Mobile** — Flutter · Dart · Riverpod · SQLite · Hive

**DevOps** — Git & GitHub · Docker · Linux · Nginx
